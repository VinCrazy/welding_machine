# Установка конденсаторной сварки термопар (односторонняя плата)

![Схема установки](docs/images/schematic.png)

Проект установки для конденсаторной сварки термопар с **односторонней печатной платой** для упрощения изготовления.

## Особенности односторонней версии:
- ✅ Все дорожки на одном слое (Bottom)
- ✅ Компоненты на противоположной стороне (Top)
- ✅ Увеличенный размер платы: 200×150 мм
- ✅ Перемычки для сложных соединений
- ✅ Упрощенное производство своими руками

## Основные характеристики
- **Тиристор:** Т134-50-8 (800А импульсный ток)
- **Конденсаторы:** 3×15000 мкФ 63В
- **Управление:** Arduino Nano
- **Максимальное напряжение:** 55В
- **Энергия импульса:** до 70 Дж
- **Размер платы:** 200×150 мм
- **Слои:** 1 (односторонняя)
