# Установка конденсаторной сварки термопар

![Схема установки](docs/schematic_images/schematic.png)

Проект установки для конденсаторной сварки и оплавления термопар с управлением от Arduino Nano.

## Основные характеристики

- **Тиристор:** Т134-50-8 (800А импульсный ток)
- **Конденсаторы:** 3×15000 мкФ 63В
- **Управление:** Arduino Nano
- **Максимальное напряжение:** 55В
- **Энергия импульса:** до 70 Дж
- **Управление импульсом:** 1-100 мс (регулируемое)

## Структура проекта

- `kicad/` — файлы проекта KiCad (схема и печатная плата)
- `docs/` — документация (BOM, инструкции по сборке)
- `src/` — исходный код для Arduino
- `resources/` — дополнительные ресурсы (даташиты, 3D модели)

## Безопасность

**ВНИМАНИЕ!** Установка работает с высоким напряжением (до 55В) и большими токами. Соблюдайте меры безопасности:
- Всегда разряжайте конденсаторы перед работой с платой.
- Не прикасайтесь к силовым шинам при включенном питании.
- Используйте изолированные инструменты и защитные очки.
- Установите защитный кожух.
